# Разработка веб-сайта интернет-магазина компьютерной техники «TECHRISE»

**Автор:** Барон Денис
**Тип проекта:** Учебная практика
**Год:** 2026

---

## 1. Актуальность темы

В современном мире электронная коммерция (e-commerce) стала неотъемлемой
частью повседневной жизни. По данным аналитических агентств, объём онлайн-продаж
ежегодно растёт на 15–20%, а сегмент компьютерной и игровой техники — один
из самых востребованных.

Рост популярности онлайн-покупок обусловлен рядом факторов:

- **Удобство** — покупка в любое время без посещения физического магазина;
- **Широкий ассортимент** — возможность сравнить десятки моделей на одной странице;
- **Экономия времени** — быстрый поиск, фильтрация и оформление заказа;
- **Доступность информации** — подробные характеристики, отзывы, рейтинги.

Особенно актуален рынок компьютерной техники: рост игровой индустрии,
удалённой работы и обучения формирует устойчивый спрос на ноутбуки,
комплектующие и периферию. Создание удобного интернет-магазина в данной
нише является востребованной и практически значимой задачей.

## 2. Целевая аудитория

Проект ориентирован на следующие группы пользователей:

| Сегмент | Описание | Потребности |
|---------|----------|-------------|
| **Геймеры** | 16–35 лет, играют в компьютерные игры | Мощные ПК, видеокарты, игровая периферия |
| **IT-специалисты** | 20–45 лет, работают с техникой | Производительные ноутбуки, мониторы |
| **Студенты** | 17–25 лет, учатся | Бюджетные ноутбуки, аксессуары |
| **Офисные сотрудники** | 25–50 лет | Офисные ПК, клавиатуры, мыши |
| **Обычные покупатели** | Все возрасты | Простота выбора, понятный интерфейс |

**Общие характеристики аудитории:**

- Активные пользователи интернета;
- Ценят удобство и скорость покупки;
- Сравнивают цены и характеристики перед покупкой;
- Используют как настольные ПК, так и мобильные устройства.

## 3. Предполагаемые функциональные возможности

### 3.1. Для покупателя

- **Каталог товаров** с разделением по категориям:
  - Компьютеры, ноутбуки, мониторы;
  - Клавиатуры, мыши, гарнитуры;
  - Геймпады, аксессуары.

- **Карточка товара** с подробной информацией:
  - Изображение, название, цена;
  - Технические характеристики;
  - Рейтинг и описание.

- **Поиск и фильтрация:**
  - Поиск по названию и категориям;
  - Фильтр по цене (диапазон, ползунок);
  - Фильтр по рейтингу;
  - Сортировка (по цене, рейтингу, названию).

- **Корзина покупок:**
  - Добавление и удаление товаров;
  - Изменение количества;
  - Автоматический расчёт суммы;
  - Сохранение корзины между сессиями.

- **Оформление заказа:**
  - Форма с личными данными;
  - Указание адреса доставки;
  - Подтверждение заказа.

### 3.2. Для администратора (перспектива развития)

- Управление товарами (добавление, редактирование, удаление);
- Управление категориями;
- Просмотр и обработка заказов;
- Управление статусами заказов.

## 4. Используемые технологии (предварительно)

### 4.1. Frontend (клиентская часть)

| Технология | Назначение |
|------------|-----------|
| **HTML5** | Структура и разметка страниц |
| **CSS3** | Стилизация, адаптивная вёрстка |
| **JavaScript (ES6+)** | Интерактивность, логика корзины |

### 4.2. Backend (серверная часть, перспектива)

| Технология | Назначение |
|------------|-----------|
| **Node.js 20** | Серверная среда выполнения |
| **Express.js** | Веб-фреймворк, REST API |

### 4.3. База данных

| Технология | Назначение |
|------------|-----------|
| **PostgreSQL 16** | Хранение товаров, заказов, пользователей |
| **SQL** | Запросы к базе данных |

### 4.4. Инфраструктура и инструменты

| Технология | Назначение |
|------------|-----------|
| **Docker** | Контейнеризация приложения |
| **Docker Compose** | Оркестрация сервисов |
| **Git / GitHub** | Система контроля версий |
| **Ubuntu** | Операционная система для разработки |
| **Make** | Автоматизация команд сборки |

### 4.5. Архитектура хранения данных

На текущем этапе данные о товарах хранятся в структуре JavaScript
(`database.js`). В перспективе планируется перенос данных в реляционную
СУБД PostgreSQL с организацией следующих таблиц:

- `categories` — категории товаров;
- `products` — товары;
- `customers` — покупатели;
- `orders` — заказы;
- `order_items` — позиции заказов.
